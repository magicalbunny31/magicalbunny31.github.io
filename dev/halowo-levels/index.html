Overview

the librarian 📔 (code name "halowo-levels") is an informational and fun app designed for Halo's Community!!

✨ features

  • the librarian 📔's original purpose were having commands related to Tide Rush 2:
    • viewing Tide Rush 2 game data straight from the discord server using the Roblox Open Cloud API
    • interacting with Tide Rush 2 through sending announcements from the discord server using the Roblox Open Cloud API
  • there are also some....let's say, "silly" features for actuallyhalowo's private discord server, examples being:
    • the ability for server members to configure the librarian 📔's avatar, banner, and statuses
    • the ability for server members to configure the server's banner and name
    • a rudimentary soundboard, or "give-the librarian 📔-an-audio-file-and-the librarian 📔-will-join-your-voice-channel-and-play-the-audio-file" sort of thing
    • a Discord-to- Guilded message bridge that uses webhooks to synchronises messages between channels in both servers
    • a commands related to the video game Gex

📜 the librarian 📔 lore

  • the librarian 📔 was first a replacement of the AvaIre app's levelling feature - after the app had shut down - for Halo's Community!!
  • it had the code name "halowo-levels" because i was unsure what to call this app originally
    • after conspiring with actuallyhalowo for a while, we decided on the name "greg. 🦎"
    • however, i decided to keep its code name as "halowo-levels" because it'd sound better when referring to it in-code, for example through github repositories
    • the code name "halowo-levels" actually refers to its origins:
      • halowo: it was designed for Halo's Community!!
      • levels: it was a replacement of the AvaIre app's levelling feature
  • then, the app was renamed to "Gregoriah 🦎" as a reference to Regretevator
    • the other server that this app was dedicated to had a Regretevator phase where most members cosplayed as characters from the game, even running parody Bluesky accounts as them
