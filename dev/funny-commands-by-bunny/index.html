Overview

funny commands by bunny 🐇 was an app which housed miscellaneous discord application commands i had created that fit none of my other apps

💬 the lore of funny commands by bunny 🐇's first command

  • the first ever command was actually me recreating 's funny little "sex" tkinter program into a discord slash command
    • the user would enter the ages of two people
    • after input, it would bring the user to the next screen....of which was never implemented and i don't really have a clue what it was meant to show
    • the age input contained some simple data validation! for example: "what the fuck bro" is displayed when one age of the user is under 18 (unsurprisingly)
  • with the help of the new slash command /sex changed up some of the features above..
    • the age input was replaced by the ability to input any two discord users
    • an option for consent, of which when consent was not given you were greeted with an EVIL SCARY message instead
    • a silly little stick-figure gif created with the discord user's avatars of....ok, just guess at this point
    • a 1-in-100 chance of spawning balls fish
  • since this was 2021, i was slightly conscious by the nsfw connotations of this command; i didn't want to host it under an app like fox kit 🐾 as it's the front-facing bot of my public server bunny furfest 🐰🐾
    • furthermore, fox kit 🐾 at that time was exclusive to bunny furfest 🐰🐾, as well as having very furry commands that i wasn't too comfortable with sharing outside of the server's safe space
  • for these reasons listed above, i created this brand new app "funny commands by bunny 🐇" to host such commands as well as being freely available to be added to any server for all users that came across it!

🪦 how funny commands by bunny 🐇 fell off

  • ..two years went by and i only ever added two more slash commands to the app:
    • one slash command simply responded with a video once invoked
    • the other slash command was a cat soyjack ("soycat") image template which lets the user input an image for the cat to be pointing at
      • in fact,this was the banner of the app - you can even see an example of it above
  • for reasons listed in the discontinuation announcement of funny commands by bunny 🐇 below, i decided to merge the app with fox kit 🐾 in 2024
    • back in 2021, you may recall that i opposed this idea!
    • nowadays, i really don't have a care in the world about my social reputation....to an extent, anyway

funny commands by bunny 🐇 is discontinued

  • due to lack of usage and extremely unoptimised code, funny commands by bunny 🐇 has been taken offline: you'll no longer be able to install this app to any servers or to yourself
    • curious as to why? the old code required me to install some external libraries on the server itself and i didn't really like that - by moving the code to fox kit 🐾 i'm able to have the commands hosted on my newer server which means that it: supports zero system dependencies for my libraries; improves wait times by a milestone; is non-blocking by using Workers; and making me very
    • plus, i couldn't justify the two commands needing a separate app (and thus using extra resources on my server to run the gateway) with the ~0.1 commands/month throughput
  • updated commands will be available elsewhere on fox kit 🐾 - you can use them in any servers with fox kit 🐾 or in fox kit 🐾's dms; but you won't be able to add it to your servers/yourself
    • /image sex user-1 user-2 consent hidden
    • /image soycat image image-url hidden
