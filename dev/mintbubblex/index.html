Overview

MintBubblex üç≠ is a simple verification gate app designed for aethen's social area

how do server members verify?

  • new server members of aethen's social area to fill a form out, which includes:
    • a secret password found in the rules channel, ensuring that new members have scouted out the server rules
    • an optional field for your nickname
    • your pronouns
    • an optional field for extra information about yourself
  • once a verification form is sent, server staff can accept or deny applications based on these responses
    • if a verification form is accepted, the new server member is granted access to the whole server
    • if a verification form is denied, the new server member will have the opportunity to send another verification form
  • this system is also inherently able to provide server raid protection, as all new members are manually screened by server staff
  • despite MintBubblex üç≠ being created in 2022, this verification gate operates similarly to discord's own rules screening and guilds features: introduced in 2023 and 2024 respectively
