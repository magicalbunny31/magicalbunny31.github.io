Overview

greg. 🦎 (code name "halowo-levels") is an informational and fun app designed for halo's community!!

📜 greg. 🦎 lore

  • greg. 🦎 was first a replacement of the avaire app's levelling feature - after the app had shut down - for halo's community!!
  • it had the code name "halowo-levels" because i was unsure what to call this app originally
    • after conspiring with ActuallyHalowo for a while, we decided on the name "greg. 🦎"
    • however, i decided to keep its code name as "halowo-levels" because it'd sound better when referring to it in-code, for example through github repositories
    • the code name "halowo-levels" actually refers to its origins:
      • halowo: it was designed for halo's community!!
      • levels: it was a replacement of the avaire app's levelling feature
  • as time went on more commands were added to greg. 🦎, for example:
    • viewing tide rush 2 game data straight from the discord server using the roblox open cloud api
    • interacting with tide rush 2 through sending announcements from the discord server using the roblox open cloud api
  • greg. 🦎 also includes some more....let's say, "silly" features for actuallyhalowo's private discord server, including:
    • a rudimentary soundboard, or "give-greg. 🦎-an-audio-file-and-greg. 🦎-will-join-your-voice-channel-and-play-the-audio-file" sort of thing
    • custom roles by letting server members input their own: custom colour, role name, and role icon
    • a discord-to- guilded message bridge that uses webhooks to synchronises messages between channels in both servers
